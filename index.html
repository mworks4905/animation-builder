Welcome to Animation Builder!

A simple and intuitive way of building animations with little CSS knowledge. Just select the base animation you would like to start with

and start building from there using the transform tools, attribute selectors, and play back timeline

to see your animation in action. Once you are done just copy and paste your code into your CSS file and enjoy!

Code

element {
animation-duration: {{ animationProps[0] }}s;
animation-name: {{ animationProps[1] }};
}
@keyframes {{ animationProps[1] }} {

{{keyframe.frameNum}}% {
{{ keyframe.transform[0] }} : {{ keyframe.transform[1] }}({{ keyframe.x}}px, {{keyframe.y }}px) {{keyframe.transform[2]}}({{keyframe.rotate}}) {{keyframe.transform[4]}} ({{keyframe.opacity}});
}

}
